COMP3005 Assignment 3 Question 1

SETUP INSTRUCTIONS:

1. Create the database:
   - Open pgAdmin
   - Right-click "Databases" -> Create -> Database
   - Name: comp3005_assignment3
   - Save

2. Run the setup script:
   - Right-click comp3005_assignment3 database -> Query Tool
   - Open database_setup.sql and copy all contents
   - Paste into Query Tool and execute

3. Install Python dependencies:
   pip install -r requirements.txt

4. Configure database connection:
   Edit app.py and update the connection parameters (host, database, user, password) in the get_connection() function.

5. Run the application:
   python app.py
